Privacy Policy

Last Updated: May 7, 2025

Important Notice: Cooltrade is purely a cryptocurrency market data analysis tool. We do not engage in any form of cryptocurrency mining and will not use your computer resources for mining operations.

1. Information Collection

Our Chrome extension Cooltrade only collects the following necessary information:

  • User account information (email address, used only for login)
  • Public market data obtained through OKX API
  • User's custom analysis parameters and settings
  • Basic usage statistics

2. Information Usage

The information we collect is strictly used for:

  • Providing cryptocurrency market data analysis services
  • Displaying real-time market information and analysis charts
  • Saving user's personalized settings
  • Improving product features and user experience

3. Data Security

We highly value your data security:

  • All API keys and authentication information are securely encrypted and stored locally
  • All data is transmitted using HTTPS encryption
  • We do not share your personal data with any third parties
  • We do not run any unauthorized operations in the background

4. Local Storage

We use the browser's local storage functionality to:

  • Save user login status
  • Store personalized analysis parameters
  • Cache market data to improve performance

5. Cookie Usage

We use cookies for the following essential purposes:

  • Authentication: To maintain your login session and ensure secure access to your account
  • Invitation System: To process and validate invitation codes for new users
  • Security: To protect against unauthorized access and maintain the integrity of our service

These cookies are strictly necessary for the core functionality of our extension and cannot be disabled without affecting the service's operation.

6. Third-Party Services

We only use the following necessary third-party services:

  • OKX API: For obtaining cryptocurrency market data
  • Google Analytics: For basic usage statistics (optional, users can choose to disable)

7. User Rights

As a user, you have the following rights:

  • View and export your personal data at any time
  • Modify or delete your account information
  • Choose to disable data statistics functionality
  • Uninstall the extension at any time, which will delete all locally stored data

8. Resource Usage

We promise:

  • Not to engage in any form of cryptocurrency mining
  • Not to run resource-intensive operations in the background
  • Only make API calls when necessary to obtain market data
  • Optimize performance to minimize system resource usage

9. Policy Updates

We may update this privacy policy from time to time to reflect product improvements and changes in regulatory requirements. Updated policies will be posted on this page and users will be notified through the extension.
